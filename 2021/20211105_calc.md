# 2021/11/05

> 次数が上がっても、考え方は同じ

満点:20点 / 目標:12点

$a$ を定数とする. $3$ 次関数 $f(x)=x^3-3x^2$ の $a \leqq x \leqq a+2$ における最大値と最小値を, $a$ の値によって場合分けして求めよ.

## ヒント・方針

<details markdown="1">
<summary>方針</summary>

- **最大値・最小値がいつ切り替わるか**を意識する.
- 定義域全体が動く場合の関数の最大・最小
    - 2次関数: 黄チャートIA 基本例題63
    - 3次関数: 黄チャートIIB 重要例題191
        - なぜかLv5表記だが, 逆難易度詐欺なのであまり難しくない

<details markdown="1">
<summary>すごいヒント(ほぼ答え)</summary>

- $f(a)$ と $f(a+2)$ の大小関係に注目
